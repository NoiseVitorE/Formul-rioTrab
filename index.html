k)

Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag ?

Para ocultar o valor digitado de uma senha na barra de endereços do navegador, o atributo "method" não é o responsável. O atributo "method" especifica como os dados do formulário serão enviados, geralmente usando os valores "GET" ou "POST". Para proteger a senha, você deve usar o atributo "method" com o valor "POST" e o atributo "type" com o valor "password" na tag correspondente ao campo de senha.


l)

Qual valor deve ser informado no atributo "method" na tag em um formulário que será usado para fazer upload de arquivos?

Para formulários que serão usados para fazer upload de arquivos, o valor do atributo "method" na tag geralmente é definido como "POST". Isso ocorre porque o método "POST" permite o envio de dados do formulário de maneira segura e eficiente, incluindo o envio de arquivos.


m)

O que as tags fieldset e legend produzem?

a tag fieldset é usada para agrupar elementos, assim como a tag label, dentro de um formulário web. Já a tag legend define uma legenda para o elemento fieldset.
